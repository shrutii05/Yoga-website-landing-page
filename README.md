# Yoga Website Landing Page

A modern and responsive Yoga Landing Page built using HTML, CSS, and JavaScript. The website features a clean design, mobile-friendly navigation, smooth animations, and an engaging user interface for yoga and wellness businesses.

## 🚀 Features

* Responsive design for all devices
* Modern and attractive UI
* Mobile navigation menu
* Smooth scroll animations with ScrollReveal
* Social media integration
* Clean and organized code structure
* Optimized layout for user engagement

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* ScrollReveal
* Remix Icons
